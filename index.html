Tic Tac Toe Game

In this project, I've created a classic game of Tic Tac Toe using TypeScript. The game features a user-friendly interface where players can take turns marking Xs and Os on a 3x3 grid. With TypeScript, I've enhanced the codebase with static typing, making it more robust and maintainable. Players can compete against each other in a friendly match, and the game dynamically updates the board and checks for a winner after each move.
